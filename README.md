## 📌 Sobre a Trilha

Programa de nivelamento em IA com duração de 8 semanas, cobrindo desde fundamentos de programação e pensamento computacional até conceitos de machine learning. O curso combina Python, Pandas, notebooks interativos no Google Colab e ferramentas de IA generativa.

**Autor:** Lucas Yudy Okuda  
**Instituição:** Alura × Santander Open Academy  
**Status:** 🟡 Em andamento (10/14 cursos concluídos)

---

## 🎯 Objetivo

- Compreender princípios de pensamento computacional, lógica e estruturas de software
- Aplicar conceitos de Python: condicionais, laços, funções, estruturas de dados
- Manipular dados com **Pandas** e **NumPy**
- Utilizar ferramentas de IA generativa: ChatGPT, Google AI Studio, Gemini e Maritaca AI
